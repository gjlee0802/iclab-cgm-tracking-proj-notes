## xDrip
https://github.com/NightscoutFoundation/xDrip

- xDrip+는 다양한 CGM(Continuous Glucose Monitoring) 센서와 블루투스를 통해 직접 연결 가능
- Dexcom G4/G5/G6, Libre, Medtrum 등 여러 센서와의 블루투스 연결을 지원하여 실시간 혈당 데이터를 수집하고 관리 가능
